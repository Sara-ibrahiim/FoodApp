
# Food Management Application

Introduction

food web that offers seamless online , it features user management roles (Admin/User), CRUD operations, Users can enjoy real-time browsing, filtering, searching .

---------------------------------------------------------------------------------------

Technologies:

React

Vite

React Bootstrap

React Router DOM

React Hook Form

React Sidebar

Axios

React Toastify

React Drag Drop Files

react-beforeunload

Context API


Business Features:

User Management Roles (Admin/User)

CRUD Operations: Easily create, update, and delete items

Search Functionality: Filter products by name or category to quickly find what you need

Favorites: Users can mark recipes as favorites

Authorization: Valid authentication for secure access




To Set Up the Project

1. Clone the Repository:
git clone https://github.com/Sara-ibrahiim/FoodApp.git


2. Install Dependencies:
npm install


3. Start the Development Server:
npm start
